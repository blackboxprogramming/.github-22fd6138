## Products

### RoadAuth — Identity & Access Management
The IAM platform built for compliance — not bolted onto it. 4 AI security agents (Sentinel, Auditor, Enforcer, Provisioner). SOC 2, HIPAA, FedRAMP out of the box. JWT + Paseto tokens, WebAuthn biometrics, OAuth2/LDAP/SAML/SCIM 2.0.

**$53.9B TAM** · Save $24K–$18M/year vs. Auth0 & Okta

### Lucidia — AI with Memory
The AI that remembers you. Universal memory layer across all AI models. Personal companion. Enterprise orchestrator. 108 local models via Ollama. Sovereign — runs on your hardware, no cloud dependency.

**$437.5B TAM** · [lucidia](https://github.com/blackboxprogramming/lucidia) · [lucidia-cli](https://github.com/blackboxprogramming/lucidia-cli)

### Quantum Framework — Real Quantum Computing
Design quantum circuits visually. Simulate them instantly. VQE, QAOA, Grover's, QFT. 1,012 verified equations. Real qudit computing.

**$7–14B TAM** · [quantum-math-lab](https://github.com/blackboxprogramming/quantum-math-lab) · [circuits.blackroad.io](https://circuits.blackroad.io)
